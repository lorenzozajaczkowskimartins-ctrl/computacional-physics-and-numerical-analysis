# Contributing Guidelines

Thank you for considering contributing to this repository.

This project is a complete mathematical and computational toolkit for
undergraduate Physics and Engineering, covering:

- Linear Algebra (symbolic and numerical)
- Vector Analysis and Vector Calculus
- Geometric Linear Algebra
- Geometric Algebra and Calculus
- Numerical Methods and Scientific Computing

## How to Contribute

### 1. Fork the Repository
Create a fork and work on a separate branch.

### 2. Suggested Contributions

Examples of valuable contributions include:

- New modules (vector calculus, differential operators, integrals)
- Geometric Algebra extensions
- Physics-based worked examples
- Visualization tools (fields, surfaces, heatmaps)
- Bug fixes and performance improvements
- Documentation and educational explanations

### 3. Code Philosophy

This repository prioritizes:

- Mathematical clarity and correctness
- Symbolic-first understanding (SymPy)
- Numerical validation (NumPy/SciPy)
- Modular and educational design

### 4. Submitting Changes

Open a Pull Request including:

- Clear description of changes
- Motivation and mathematical context
- Example outputs or notebooks if relevant
